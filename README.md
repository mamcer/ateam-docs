# ATeam

Built with Sphinx

    sudo apt-get install python-sphinx
    sudo apt-get install python-pip
    pip install -U sphinx_rtd_theme

    sphinx-build -nW -b html docs docs/_build/html
    
    cd docs/_build/html
    python -m SimpleHTTPServer 8080 .
